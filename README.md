# Excel Sales Dashboard

## Project Overview

This project demonstrates data cleaning, analysis, and dashboard creation using Microsoft Excel (or Google Sheets) with the Superstore Sales Dataset. The dashboard provides key business insights through interactive visualizations, Pivot Tables, Charts, and Conditional Formatting.

## Objective

The main objective of this project is to:

* Analyze sales and profit performance.
* Visualize business trends over time.
* Identify high-performing and low-performing categories, regions, and products.
* Build an interactive dashboard for decision-making.

## Dataset

**Dataset Used:** Superstore Sales Dataset

### Key Columns

* Order ID
* Order Date
* Customer Name
* Segment
* Region
* Category
* Sub-Category
* Sales
* Quantity
* Profit

## Data Cleaning Process

The following data cleaning steps were performed:

1. Removed duplicate records.
2. Checked and handled missing values.
3. Formatted date fields correctly.
4. Converted Sales and Profit columns to currency format.
5. Validated data consistency and removed errors.

## Dashboard Features

### KPI Cards

* Total Sales
* Total Profit
* Total Orders
* Average Sales

### Visualizations

#### 1. Profit by Category

* Clustered Column Chart
* Compares profitability across product categories.

#### 2. Sales by Region

* Bar Chart
* Displays regional sales performance.

#### 3. Monthly Sales Trend

* Line Chart
* Shows sales growth and seasonal patterns over time.

### Interactive Filters

Slicers were added for:

* Category
* Region
* Segment
* Ship Mode

### Conditional Formatting

Used to highlight:

* High-profit products (Green)
* Low-profit or loss-making products (Red)
